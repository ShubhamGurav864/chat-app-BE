🚀 Chat Application Backend (Spring Boot)
A scalable backend system for a real-time chat application built using Java + Spring Boot.
This project provides REST APIs for user authentication and messaging, with MongoDB for persistence and Redis for caching.

📄 Swagger API Docs (Interactive):
https://chat-app-be-psfy.onrender.com/swagger-ui/index.html

📌 Features
✅ User Registration & Login
✅ Send and Retrieve Messages
✅ RESTful API Design
✅ MongoDB Integration (NoSQL DB)
✅ Redis Integration (Caching / Realtime support ready)
✅ Dockerized Deployment
✅ Deployed on cloud (Render)

🛠️ Tech Stack
Backend: Java, Spring Boot
Database: MongoDB
Cache / Realtime Support: Redis
Build Tool: Maven
Deployment: Docker + Render
API Testing: Swagger UI
